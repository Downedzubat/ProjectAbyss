# PROJECT: ABYSS
## How to add characters
To add a character, navigate to the following folder within the engine

Content/Blueprints/Characters

in here you will see all the character blueprints and one called FighterTemplateCharacter. 

Right Click fighter template character and select create child blueprint class

Name the new class and open it.

In here you will be able to edit the following:

NAME
HUD ICON
MOVELIST
INPUT LIST
MESH
HITBOX



DO NOT MAKE CHANGES WITHIN FIGHTERTEMPLATECHARACTER, ELSE YOU RISK ACCIDENTALLY REMOVING ALL OVERRIDES AND LOGIC IN ALL CHARACTERS
